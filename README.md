# Bicycle Ki Calculator

Browser-based Ki calculator for fluorescence polarization competitive binding data.

## Usage

1. Open the GitHub Pages site.
2. Upload an Excel workbook using the same template layout.
3. Enter protein concentration, tracer concentration, and tracer Kd.
4. Run the calculation and download the result workbook.

The Excel file is processed locally in the browser. It is not uploaded to GitHub or any server by this page.

## Files

- `index.html`: main calculator page.
- `vendor/xlsx.bundle.js`: local Excel parser used by the page.
